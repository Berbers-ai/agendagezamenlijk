Agenda instellen

Verbind Supabase voor realtime sync tussen al je apparaten. Je kunt dit ook overslaan en alleen lokaal werken.

Supabase instellen (gratis, ~5 min):
  1. Ga naar supabase.com → nieuw project aanmaken
  2. Ga naar Project Settings → API
  3. Kopieer de Project URL en anon public key
  4. Ga naar SQL Editor en voer dit uit:
    create table events (id text primary key, data jsonb);
    create table todos (id text primary key, data jsonb);
    alter table events enable row level security;
    alter table todos enable row level security;
    create policy "open" on events for all using (true) with check (true);
    create policy "open" on todos for all using (true) with check (true);
  5. Plak de URL en key hieronder
✓ Gesynchroniseerd
